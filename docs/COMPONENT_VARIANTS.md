# Component Variants

Some Ant Design components expose official subcomponents with different API sections. S2.2 represents those as separate registry variants when their editing contracts differ.

Current implemented variants:

- `FloatButton`
- `FloatButton.Group`
- `FloatButton.BackTop`

Future variants should follow the same rule for Dropdown.Button, Form.Item, Form.List, Table.Column, Tabs items, and Collapse items.
